
# 键盘

## hideKeyboard

隐藏键盘

**入参**

| 参数      | 类型   | 说明                |        支持平台               |
| --------- | ------ | ------------------|------------------------ |
| success | Function | 接口调用成功的回调函数 |  微信
| fail | Function | 接口调用失败的回调函数 |  微信
| complete | Function | 接口调用结束的回调函数（调用成功、失败都会执行） |  微信
